# Matrices

Консольное Java-приложение для работы с матрицами: сложение, вычитание, умножение и вычисление определителя.

## Возможности

- Загрузка двух матриц из текстовых файлов
- Сложение и вычитание матриц
- Умножение матриц
- Вычисление определителя (методом Гаусса)
- Логирование ошибок в консоль и файл `logs/matrix.log`

## Требования

- JDK 17–24 (версии выше 24 не поддерживаются из-за используемой версии Gradle 8.14)

## Формат файла с матрицей

Элементы строки записываются через пробел, каждая строка матрицы — на новой строке файла. Например:

```
1 2 3
4 5 6
7 8 9
```

## Сборка проекта

В корневом каталоге проекта выполните:

```
.\gradlew.bat build
```

Собранный `.jar`-файл появится в каталоге `build/libs`. Примеры файлов матриц лежат в каталоге `examples` в корне проекта.

## Запуск

```
java -jar Matrices.jar file1.txt file2.txt
```

где `file1.txt` и `file2.txt` — пути к файлам с матрицами (аргумент 1 и аргумент 2 соответственно).

После запуска программа выводит обе загруженные матрицы и предлагает выбрать операцию:

```
************************************************************
Введите операцию
1 - сложение матриц
2 - разность матриц (вычитает из args[0] матрицу args[1])
3 - произведение матриц (умножает args[0] справа на args[1])
4 - вычисление определителя одной из матриц
Любой другой символ для выхода
************************************************************
```

При выборе пункта **4** дополнительно нужно указать, для какой из двух матриц вычислить определитель.

Для выхода из программы введите любой символ, отличный от `1`–`4`.

## Логирование

Ошибки выполнения (например, несовпадение размеров матриц или неверный путь к файлу) выводятся в консоль и дополнительно записываются в файл `logs/matrix.log`.

## Ограничения

- Не поддерживаются версии Java выше 24 (используется Gradle 8.14)
